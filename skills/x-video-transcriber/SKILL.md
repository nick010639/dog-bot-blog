# X 视频帖串解析技能

这个技能能够帮你深度挖掘 X (Twitter) 上的长视频内容，通常这些内容以帖串（Thread）形式发布，包含了完整的视频文字脚本或核心摘要。

## 技能原理

1.  **元数据提取**：利用 `x-api-fetch` 或 `playwright-bot-bypass` 获取视频推文的元数据（如 OEmbed JSON）。
2.  **多层级文本抓取**：
    *   **推文标题**：从页面 HTML 源码中提取。
    *   **核心文本**：抓取元数据中的 `description` 字段。
    *   **时间轴解析**：识别帖串中不同的推文，并根据视频时间戳关联内容。
3.  **智能总结**：利用 LLM 结合抓取到的文本和时间轴，生成结构化的中文总结。

## 基础用法

在对话中直接告诉 Bot 该推文的链接即可：

```text
请解析并总结这个视频帖串：https://x.com/username/status/123456789
```

## 限制说明

*   **环境依赖**：此技能需要 `skills/x-api-fetch` 或 `playwright-bot-bypass` 技能的支持。
*   **无法转录音频**：目前配置不支持直接从视频提取音频流并转录（需要 Whisper 或类似环境），因此仅能提取现成的文字脚本。
*   **API 限制**：X 平台有严格的反爬虫机制，高频抓取可能导致 IP 被封禁。

## 高级用例

1.  **"大佬技能化"**：将视频中的干货转化为你的 Agent Skill。
2.  **时间轴笔记**：根据视频时间戳，将长内容拆分为不同章节的 Markdown 笔记。
3.  **元数据归档**：定期将提取的视频脚本和元数据存入你的知识库。
