# ✍️ Writer Skill (Zack Style)

This skill formalizes the "plain-human" writing style preferred by Zack. It is designed to be iterated upon as the bot's writing capabilities evolve.

## 核心写作原则 (Core Principles)

1.  **平实叙述 (Plain Narrative)**: 像在咖啡馆跟朋友聊天一样。去掉翻译腔，去掉“AI味”。
2.  **拒绝华丽 (No Fluff)**: 严禁使用夸张的形容词和成语（如“爆炸性”、“史诗级”、“深水炸弹”）。
3.  **随性连接词 (Casual Connectors)**: 善用“他说”、“他说”、“我觉得”等口语引导词。
4.  **极简点缀 (Minimalist Emojis)**: 全文仅在**开头**和**结尾**处各保留一个表情符号。
5.  **自然排版 (Natural Layout)**: 每一段就是一个完整的观点，不用过分强调 1/5 这种编号，让段落自然流动。

## 格式规范 (Formatting)

- **推文串 (Threads)**: 优先采用连贯的段落叙述，每段代表一个推文。
- **链接**: 保持原始 URL，放在新的一行，避免 Markdown 语法。
