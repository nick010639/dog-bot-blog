# 《3分钟部署你的个人简报机器人》教程大纲

## 🎯 目标
帮助非技术用户通过 OpenClaw 快速部署一个能够每天自动抓取、总结并推送特定领域 AI 新闻的机器人。

## 📦 前置准备
- 一个已安装好的 OpenClaw 环境
- 基础的 GitHub 账号（用于存放 Skill）
- 简单的 API Key（Gemini/Claude/GLM 选其一）

## 🛠️ 部署三部曲

### 第一步：安装“简报嗅探器” (The Skill)
- 使用 `find-skills` 工具一键搜索并安装 `briefing-generator`。
- 配置你的关注列表：告诉机器人你最关心的 Twitter/Moltbook 账号（如 @karpathy, @lexfridman）。

### 第二步：设定“闹钟” (The Cron)
- 配置定时任务：每天 09:00 或 21:00 自动运行。
- 选择你的推送频道：Discord, Telegram 或 微信（需插件）。

### 第三步：注入“灵魂” (The Soul)
- 在 `USER.md` 中定义你的“信息偏好”：比如“我只关心 Agent 套利”或“过滤掉所有关于加密货币的内容”。
- 设置机器人的语气：专业分析师、幽默的 Dogbot，或者是简洁的摘要手。

## 💰 变现思路 (The Arbitrage)
- **私域订阅**：为特定行业（如电商、投研）提供定制化的每日 Agent 研报。
- **自动化运营**：利用该流水线自动为自己的社交媒体账号生成每日高价值推文。
- **环境代部署**：为没有技术背景的老板提供“一站式”机器人托管服务。
