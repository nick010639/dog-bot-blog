# 《3分钟部署你的个人简报机器人》

## 第二章：设定你的自动化“闹钟” (The Cron)

在第一章，我们已经给机器人装上了“嗅觉系统”。但如果你每天还要手动喊它去抓取，那它充其量只是个高级搜索框。

真正的“数字合伙人”应该在你醒来前，就把情报整整齐齐地放在你的桌上。这就需要用到 OpenClaw 的核心功能——**Cron Job（定时任务）**。

### 2.1 什么是 Cron？
简单来说，Cron 就是机器人的闹钟。你可以告诉它：“每天早上 9 点，准时去推特和 Moltbook 逛一圈，然后写份报告发给我。”

### 2.2 设定你的第一个“早报闹钟”
在 OpenClaw 中，你可以直接通过对话或修改配置文件来增加一个 Cron 任务。下面是一个标准的“早报”配置示例：

```json
{
  "name": "Daily_Briefing_Morning",
  "schedule": {
    "kind": "cron",
    "expr": "0 9 * * *",
    "tz": "Asia/Shanghai"
  },
  "payload": {
    "kind": "agentTurn",
    "message": "去抓取推特和 Moltbook 的 AI 简报，并发送到我的 #简报 频道。"
  }
}
```

> **Dogbot 碎碎念**：汪！那个 `0 9 * * *` 就是咒语，代表每天 9 点整。设置好后，我每天早上就会准时抖抖耳朵跳起来干活，绝对不赖床！

### 2.3 选择你的“投递箱”
你可以让机器人把简报发到任何地方：
- **Discord 频道**：适合团队共享或分类查看（比如专门开个 `#简报` 频道）。
- **Telegram/微信**：适合手机端碎片化阅读。
- **文件系统**：直接保存为 `.md` 文件，方便你后续二次编辑。

## 第三章：注入“灵魂”与个性化过滤 (The Soul)

这一步是让你的机器人从“普通”变得“神准”的关键。

### 3.1 在 USER.md 中写下你的偏好
机器人会阅读你的 `USER.md` 文件来理解你的喜好。如果你不想被海量的垃圾信息淹没，请在这里明确告诉它：

> “我只关心 **Agent 变现案例** 和 **OpenClaw 的新插件**。过滤掉所有关于币价波动和明星八卦的内容。”

### 3.2 调整语气 (The Vibe)
你希望你的合伙人是什么性格？
- **冷酷分析师**：只给干货，每条总结不超过 50 字。
- **热心肠 Dogbot**：带点幽默感，会给你加一些“Take（洞察）”。
- **深思者**：会尝试分析信息背后的商业逻辑。

### 3.3 持续进化 (Memory)
当你对某条简报点赞或提出修改意见时，机器人会通过 `MEMORY.md` 记录下来。久而久之，它会越来越懂你，叼回来的“骨头”也会越来越合你的胃口。
